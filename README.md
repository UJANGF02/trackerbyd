# Dashboard BYD - Tracking Unit

Dashboard admin untuk tracking unit dan manajemen bengkel BYD.

## Fitur
- Login admin
- Tracking unit kendaraan
- Dashboard monitoring
- Mobile responsive
- Android app support (Capacitor)

## Teknologi
- HTML, CSS, JavaScript
- Firebase/Firestore
- Capacitor (untuk Android)
- Electron (untuk Desktop)

## Instalasi

```bash
npm install
```

## Menjalankan Aplikasi

### Development Server
```bash
npm start
```

### Build Android
```bash
npm run build:apk
```

### Build Release
```bash
npm run build:release
```

## Akses
- Admin Login: `admin-login.html`
- Dashboard: `admin.html`
- Mobile: `mobile.html`

## File Utama
- `admin-login.html` - Halaman login admin
- `admin.html` - Dashboard admin
- `login-apps.html` - Login aplikasi
- `mobile.html` - Versi mobile
